Python Project Launcher - Init.py

Overview

This is a python project launcher. It uses a combination of file handling methods to create and write python file, when requested. Because I am opposed to documentation, there is little to no documentation, aside from at the start. Each file is stored in a docstring, and is written with the create() function. This file can and will work locally, no matter what filepath it is in. This is because of the ntpath, sys, and os python modules.

Currently there is only one project stored:

Command Line Interface

This is the only completed project as of 2/15/2026. It was made to model a cisco switch from the networking simulation software 'Packet Tracer'. Despite the name, it is not a traditional command line interface. Instead of gaining input from a command prompt, it uses the output terminal to gather information.

How do you create this project? When you run the project (init.py), you will be prompted for input. Enter the words 'Command Line Interface'.
Project size: 24KB

How to access?

Copy the code that is down , and make sure that it is formatted as it is in the window that shows the code. If it is different, change it to be correct. If the indentation looks different, that is because of how the HTML is formatted, not the text. Make sure that if the code says a = __ the code that you run says the same.
Total project size: 24KB+ (More storage will be taken up when projects are created)
